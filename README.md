fabric网络环境搭建的例子，所使用的yaml和各种配置。

# e2e_cli

# kafka

参考社区[fabric-sample-with-kafka](https://github.com/keenkit/fabric-sample-with-kafka)，在其基础上，修改为三组织三通道配置，只要orderer集群节点对容器外暴露接口。
